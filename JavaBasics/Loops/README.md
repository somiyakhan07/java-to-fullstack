# Loops in Java

This folder contains the programs I wrote while learning loops in Java. It includes examples and practice problems to understand different looping concepts and improve logical thinking.

### Topics

* `for` Loop
* `while` Loop
* `do-while` Loop
* Enhanced `for` Loop (for-each)
* Nested `for` Loop
* Nested `while` Loop
* Loop Control Statements (`break` and `continue`)
* Pattern Printing
